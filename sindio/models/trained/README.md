# Trained models (PyTorch / Safetensors)

Place your trained model checkpoints here:

- `urban_stress_v1.pth` — Multi-modal stress prediction (GIS + mobility + utility)
- `mobility_v2.pth` — Mobility and transit demand forecasting
- `water_demand_v1.pth` — Water grid demand prediction

Files in this directory are git-ignored.
Use `dvc pull` or download from your model registry.
